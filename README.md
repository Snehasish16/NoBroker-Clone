#  NoBroker Website Clone (HTML & CSS)

This project is a **static clone of the NoBroker homepage** created using **pure HTML and CSS only**.  
The goal of this project is achieve **frontend layout building, and real-world UI cloning** without using any frameworks or JavaScript.


## preview

##  Features Implemented

- Clean and structured HTML
- Modern CSS (Flexbox & Grid)
- Real-world UI sections cloned:
  - Header & Navigation
  - Property search sections
  - Services section (Buy / Rent)
  - Multi-column footer links
  - CTA sections (Find Property / List Property)
- Pixel-close UI inspired by NoBroker


##  Technologies Used

- **HTML5**
- **CSS3**
